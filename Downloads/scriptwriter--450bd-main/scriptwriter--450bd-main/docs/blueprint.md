# **App Name**: Screenwriter Studio

## Core Features:

- Script Editor: A real-time collaborative script editor with automatic formatting.
- Character and Scene Management: Tools to organize characters and scenes within a script.
- Collaboration: Real-time collaboration features with commenting and version history.
- Media Integration: Ability to upload and manage media assets within the project.
- Script Formatting Tool: Tool that formats screenplay content based on industry standard conventions and downloads it to various different file types
- AI Script Analysis: AI tool that analyzes a script for plot holes, pacing issues, or character inconsistencies.
- User Authentication and Storage: Secure user authentication and script storage using Firestore.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) for a professional and reliable feel.
- Background color: Light gray (#F0F4F8) to ensure readability and reduce eye strain during long writing sessions.
- Accent color: Bright yellow (#FFC857) for highlighting important elements and calls to action.
- Body and headline font: 'Inter' for a modern and clean interface, ensuring readability for both headings and body text (sans-serif).
- Use minimalist and consistent icons to represent actions and tools.
- A clean and organized layout that mimics industry-standard screenwriting software, focusing on usability and reducing clutter.
- Subtle animations for a smooth user experience (e.g., when saving or updating scripts).
